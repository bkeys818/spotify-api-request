    Parameters

    • token: Token | string

      A valid user access token or your client credentials.

getAlbumTracks

  • getAlbumTracks(token: Token | string, id: string, options?: { limit?: number; market?: string; offset?: number }): Promise<PagingObject<SimplifiedTrackObject, "album’s tracks">>

  • Get Spotify catalog information about an album’s tracks. Optional parameters can be used to limit the number of tracks returned.

    Parameters

    • token: Token | string

      A valid user access token or your client credentials.

    • id: string

      The Spotify ID of the album.

    • Optional options: { limit?: number; market?: string; offset?: number }
      • Optional limit?: number

        The maximum number of tracks to return. Default: 20. Minimum: 1. Maximum: 50.

      • Optional market?: string

        The market you’d like to request. Synonym for country.

      • Optional offset?: number

        The index of the first track to return. Default: 0 (the first object). Use with limit to get the next set of tracks.

    Returns Promise<PagingObject<SimplifiedTrackObject, "album’s tracks">>

    An album object in JSON format.

getMultipleAlbums

  • getMultipleAlbums(token: Token | string, ids: string[], options?: { market: string }): Promise<{ albums: (AlbumObject | null)[] }>

  • Get Spotify catalog information for multiple albums identified by their Spotify IDs.

    Parameters

    • token: Token | string

      A valid user access token or your client credentials.

    • ids: string[]

      A comma-separated list of Spotify IDs for albums. Maximum: 20 IDs.

    • Optional options: { market: string }
      • market: string

        An ISO 3166-1 alpha-2 country code orstring from_token. Provide this parameter if you want to apply Track Relinking.

    Returns Promise<{ albums: (AlbumObject | null)[] }>

    Objects are returned in the order requested. If an object is not found, a null value is returned in the appropriate position. Duplicate ids in the query will result in duplicate objects in the response.
