searchForItem

  • searchForItem<T>(token: Token | string, options: { include_external?: "audio"; limit?: number; market?: string; offset?: number; q: string; type: T }): Promise<ResponseType<T>>

  • Get Spotify Catalog information about albums, artists, playlists, tracks, shows or episodes that match a keyword string.

    Type parameters

    • T: keyof ResponseObject | keyof ResponseObject[]

    Parameters

    • token: Token | string

      A valid access token from the Spotify Accounts service: see the Web API Authorization Guide for details.

    • options: { include_external?: "audio"; limit?: number; market?: string; offset?: number; q: string; type: T }
      • Optional include_external?: "audio"

        If audio is specified the response will include any relevant audio content that is hosted externally.

        By default external content is filtered out from responses.

      • Optional limit?: number

        Maximum number of results to return.

        Default: 20

        Minimum: 1

        Maximum: 50

        Note: The limit is applied within each type, not on the total response.

        For example, if the limit value is 3 and the type is ['artist', 'album'], the response contains 3 artists and 3 albums.

      • Optional market?: string

        An ISO 3166-1 alpha-2 country code or the string "from_token".

        If a country code is specified, only content that is playable in that market is returned.

        Note:

        • Playlist results are not affected by the market parameter.

        • If market is set to from_token, and a valid access token is specified in the request header, only content playable in the country associated with the user account, is returned.

        • Users can view the country that is associated with their account in the account settings. A user must grant access to the user-read-private scope prior to when the access token is issued.

      • Optional offset?: number

        The index of the first result to return.

        Default: 0 (the first result).

        Maximum offset (including limit): 1,000.

        Use with limit to get the next page of search results.

      • q: string

        Search query keywords and optional field filters and operators.

      • type: T

        A list of item types to search across. Search results include hits from all the specified item types.

    Returns Promise<ResponseType<T>>

    For each type provided in the type parameter, the response contains an array of {@link ArtistObject artist objects} / {@link SimplifiedAlbumObject simplified album objects} / {@link TrackObject track objects} / {@link SimplifiedShowObject simplified show objects} / {@link SimplifiedEpisodeObject simplified episode objects} wrapped in a {@link PagingObject paging object}
