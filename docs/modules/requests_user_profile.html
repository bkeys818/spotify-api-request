getCurrentUserProfile

  • getCurrentUserProfile(token: Token | string): Promise<PrivateUserObject>

  • Get detailed profile information about the current user (including the current user’s username).

    Parameters

    • token: Token | string

      A valid access token from the Spotify Accounts service: see the Web API Authorization Guide for details. The access token must have been issued on behalf of the current user.
      Reading the user’s email address requires the user-read-email scope; reading country, product subscription level and explicit content settings requires the user-read-private scope. See Using Scopes.

    Returns Promise<PrivateUserObject>

    A {@link PrivateUserObject user object}.
    Important! If the user-read-email scope is authorized, the returned JSON will include the email address that was entered when the user created their Spotify account. This email address is unverified; do not assume that the email address belongs to the user.
